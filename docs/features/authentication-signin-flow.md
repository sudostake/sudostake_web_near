# Wallet sign-in flow

## Steps
1. Click `Connect Wallet` (or borrower CTA on landing).
2. Choose wallet provider in Wallet Selector.
3. Approve account connection in wallet.
4. Return to app with connected account shown in nav.

## If the flow is canceled
- No transaction is sent.
- You remain signed out.
- You can retry at any time.

## After sign-in
- Dashboard is available.
- Vault and lender actions appear based on role and vault state.
