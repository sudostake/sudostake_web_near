# Create a vault

## TL;DR
- Time: about 2 minutes.
- Requirement: connected wallet with enough NEAR for the vault creation deposit (currently `10 NEAR`).
- Result: a new vault appears in your dashboard.

## Before you start
- Sign in with a supported wallet.
- Confirm the network shown in the header.
- Keep extra NEAR for follow-up actions (deposit, delegation, request creation).

## Steps
1. Open `/dashboard`.
2. Click `New vault` in the header area, or `Create vault` from the vault tools area.
3. Review the one-time cost in the dialog.
4. Click `Create Vault` and approve in wallet.
5. Wait for indexing to complete.

## What you should see
- The new vault appears in `Vaults`.
- Opening the vault page shows owner tools and liquidity request controls.

## Common issues
- Wallet rejected: no vault is created.
- Insufficient NEAR: top up wallet and retry.
- Vault missing after success: refresh or retry indexing.
